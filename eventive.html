Evaluate User Requirements

A fully automated online intake form that allows students, faculty, staff, and other guests the ability to submit event requests in a consistent, trackable manner. My role for this project is to focus on the user interface, user experience of the website as well as create high fidelity mockups from scratch using the program Sketch. Since Eventive is an in-house system website, I have the ability to work side-by-side with the programmers, users, and stakeholders. I have the chance of interview the stakeholders to learn what they need and what they want. I also observe the users to experience how the current process of submitting an event request is like to get a better understanding of what the event planners are thinking while planning an event.
