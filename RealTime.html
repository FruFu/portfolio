UCI Real-Time Ops Web (Redesign)

SCES Real-Time Ops is a web that has been use by all the managers in the UCI Student Center & Event Services. The purpose of this app is to automate and show the Student Center shift report data. Ensure the voice of the customer is consistently considered in strategic action items and key requirements of product and process design.

Period

March 2019 - June 2019


Methods

User & Stakeholder Interviews

Sketching

Wireframes

Prototyping


Project Type

Work Project


Tools

UI/UX Design

Wireframing

Evaluate User Requirements




Problems

Information Overload

Lots of Unused Space

Zero Flexibility (No filter, No user control, No search option)




Solutions

A redesign of the entire website that follows Nielson’s 10 heuristic evaluation. Also conducted interviews with potential users to obtain detailed information about how they currently use this app, what they currently like/don’t like about it, and what kind of struggle they encounter while using the app.
