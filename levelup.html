Level Up Program

A web-based application where student workforce can log in to see their progress toward the next level of employment and administrators can log in to track and assess learning objectives and training activities.

Understand Current Process

Each person has a document

Every student receives a personalized Leveling Document that they can view online. The students’ progress and scores are recorded and reported automatically.

Document records all trainings

Supervisors and students can initiate periodic check ins with one another to measure progress and set realistic training expectations

Levels up when fulfills all expectations

Supervisor levels up the student when they have demonstrated the knowledge and experience outlined in the checklist

The Problem

200 students = 200 individual documents

  • There are over 200 documents to update (1 per student) whenever the template for a LevelUp document is changed
    • For example, when the Operations student staff unit LevelUp template is changed, you have to manually go through and make each student’s LevelUp profile match the template.
  • There is currently no way to efficiently sort or edit the information about LevelUp profiles.
  • There is no way to do bulk account creation for batches of student employees.
    • For instance, when 50 students are hired all at once, you have to manually go through and create each individual account, rather than instantiating the accounts all at once.
  • The self-reporting system for student employees is not user friendly.

The Goal

Web dashboard that connects to the database

  • Need to Have (i.e., What does it take to get the product off the ground?)
    • Student Dashboard Profile
      Basic Functionality: To log in, see a profile, and have it be at least as good as the Google Form except in our system
    • Admin Dashboard
      Basic Functionality: To log in, be able to see specific student dashboards, see department stats, create/edit templates
  • Should Have (i.e., Features that make the product valuable)
    • Ranking up, including for unusual rankups like L1A - L1B
    • Quizzes integrated into the system (the ones that are currently Google Forms)
    • Improve the design of the Progress Reports Chart (the Information Tracking stuff)
    • Show Status of LevelUp for a Student
